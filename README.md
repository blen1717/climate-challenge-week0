# Climate Analysis for COP32

## Overview
This project analyzes climate data from 5 African countries (Ethiopia, Kenya, Nigeria, Sudan, Tanzania) from 2015-2026 to support Ethiopia's preparation for hosting COP32.

## Key Findings
- Sudan is most climate-vulnerable: 2,694 extreme heat days (>35°C)
- Ethiopia is least vulnerable: 0 extreme heat days, coolest temperatures (16.1°C average)
- Sudan shows highest temperature variability (std 4.7°C)

## How to Run
1. Open in Google Colab
2. Upload the 5 country CSV files
3. Run all cells

## Requirements
- Python 3.8+
- pandas
- numpy
